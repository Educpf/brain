
# Purpose

Have a way to easily have my ideas online making the process of documenting and writting down knowledge for long time memory easy!

## Stack

| Tool/Libary | Feature | 
| --- | -- |
| marked.js / markdown-it | Markdown -> HTML | 
| Prism.js or highlight.js | syntax highlight code |
| gray-matter | parsing formatter | 
|chokidar(node.js) | hot reload(file watching) |
|Lunr.js | search |
| Tailwind.css | styling |
